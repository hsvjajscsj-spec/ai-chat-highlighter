# Chat Highlight - GitHub 仓库引流配置参数 (Repo Settings)

为了最大化 GitHub 站内搜索以及 Google 搜索引擎的流量拦截，请在创建仓库后，直接复制以下参数进行配置。这些词汇结合了高频 SEO 搜索意图。

## 1. 仓库名称 (Repository Name)

不要只用品牌名。必须用“动作 + 平台 + 结果”的组合。

**推荐（命中率最高）：**
```text
chatgpt-export-markdown
```

**备选：**
- `ai-chat-highlighter`
- `export-chatgpt-conversations`
- `chatgpt-to-notion`
- `save-gemini-history`

---

## 2. 仓库简介 (Description)

这是一段极其重要的 1-2 句话，必须直接展示在仓库名称下方，并附带短链接。

**英文原版 (推荐使用，吸引全球流量)：**
```text
🚀 The ultimate AI chat highlighter. Export ChatGPT & Gemini conversations to flawless Markdown/Notion with perfect code formatting. ⬇️ Install Extension
```

**若必须用中文：**
```text
🚀 专为 AI 对话打造的浏览器高亮扩展。完美保留代码格式，一键提取 ChatGPT/Gemini 记录至 Markdown 或 Notion。⬇️ 点击安装插件
```

---

## 3. GitHub 网站网址配置 (Website)

在仓库右侧的 `About` 区域，有一个填写 URL 的地方。
**请一定要填 Chrome Web Store 的链接，直接引流到下载页，而不是官网。**

```text
https://chromewebstore.google.com/detail/chat-highlight-auto-chatg/ffnhgclfemimnnbkbhebjbobiipgkgbk
```

*(或者如果你想用一个短链接追踪，也可以使用官网的重定向链接)*

---

## 4. 仓库标签 (Topics / Tags)

GitHub 允许添加至多 20 个 Topics（标签）。这是站内搜索推荐的核心！！请务必在仓库界面的 `About` 旁点击齿轮（⚙️）添加以下所有标签：

**核心流量词：**
- `chatgpt`
- `gemini`
- `openai`

**功能动作词：**
- `export-chatgpt`
- `markdown`
- `web-highlighter`
- `code-formatting`
- `syntax-highlighting`

**生态关联词 (蹭势能)：**
- `notion`
- `obsidian`
- `pkm` (Personal Knowledge Management)
- `second-brain`

**技术属性词：**
- `browser-extension`
- `chrome-extension`
- `productivity-tools`
- `local-first`
- `privacy-focused`
