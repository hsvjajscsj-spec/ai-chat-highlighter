# Changelog

All notable changes to **Chat Highlight** will be documented in this file.

## [v1.0.3] - 2026-03
### Fixed
- 解析与导出修复：解决获取页面元素时偶现的元数据文件格式解析错误。
- 优化基于最新 DOM 结构的抓取鲁棒性。

## [v1.0.2] - 2026-03
### Added
- 知识库与文档升级：全面上线完整版的图文 User Guide 用户指南，帮助新用户快速掌握 Cloud Viewer 及 Chat Note 模式的高阶技巧。
- 前端官网交互与图库更新，包括全新的演示动图与架构示意图。

## [v1.0.1] - 2026-03
### Added
- 发布 Chat Highlight 初始独立站与文档支持的基础设施。
- 上线完整的 Pricing 订阅逻辑及本地存储机制架构。

## [v1.0.0] - 2026-03
### Initial Release
- **🚀 AI-Native Parsing**: 正式支持 ChatGPT 与 Gemini 的专属提取算法，完美区分 `Prompt` 和 `Answer`。
- **💻 Flawless Code Export**: 攻克了代码块复制难题，一键无损导出为保留语法级高亮的 Markdown。
- **🌳 Tree-View Sidebar**: 推出针对超长 AI 对话的侧边结构化大纲视图 (Live TOC)。
- **🔒 Privacy First**: 确定默认核心数据仅基于浏览器本地 (`chrome.storage.local`) 存储，确保极客用户的机要敏感性。
- **☁️ Cloud Viewer**: Beta 试用版云端看板上线，赋予跨设备资料检视及 Tag/图库管理的能力。
